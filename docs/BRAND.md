# BRAND — Teranova Group

## Идентичность
B2B-площадка доверенного брокериджа: связываем проверенных корейских производителей
с покупателями мира и сопровождаем сделку целиком. Бренд — премиальный, сдержанный,
«дорогой», архитектурный. Совместный бренд **AIA Group Ltd и Teranova Group Ltd**.

## Цвета (фирменные, из логотипа)
| Роль | HEX |
|------|-----|
| Тёмно-синий (основа/фон) | `#0b1b35` |
| Тёмно-синий 2 (панели/слои) | `#0e2243` |
| Кобальт (акцент, CTA, ссылки) | `#2563d4` |
| Кобальт-light (hover) | `#3f7ae0` |
| Серебро светлое (текст на тёмном) | `#EDF1F6` |
| Серебро | `#CBD4E0` |
| Приглушённое серебро (подписи) | `#92A0B5` |

- Серебро/платина = ощущение «дорого». Кобальт = акцент по минимуму (не заливать им фоны).
- Большие заголовки героев — «металлический» серебряный градиент (linear-gradient + background-clip:text).
- Никакого неона и кислотных цветов.

## Логотип
- Файлы в `/public/brand/`: `teranova-logo-light.svg` (на тёмном фоне),
  `teranova-logo-dark.svg` (на светлом), иконки `teranova-icon-*` (favicon/мобильное).
- Мотив — ромб из двух треугольников = «две стороны, которые встречаются» (мост/обмен).
  Можно использовать как крупный графический элемент (см. IMAGERY.md).
- Не растягивать, не перекрашивать произвольно, беречь свободное поле вокруг.

## Шрифт
- **Manrope** (Google Fonts), веса 300–800. Геометрический — в одном духе с логотипом.
- Заголовки — 700–800, плотный трекинг; текст — 400–500. `display=swap`.

## Тон
Премиально и по делу. Уверенно, без «воды» и маркетингового пафоса. Доверие — через факты
и процесс. Подробнее — `.skills/copywriting/SKILL.md`.
