# ТЗ: настройка форм связи (Web3Forms)

На сайте две формы: **«Под заказ»** (главная, секция `#ondemand`) и **«Тендеры»** (`/tenders`).
Обе отправляют заявки через бесплатный сервис **Web3Forms** на вашу почту. В коде всё готово —
чтобы письма реально приходили, нужно один раз задать ключ в Cloudflare.

## Как это работает
- Форма делает POST на `https://api.web3forms.com/submit` с вашим `access_key`.
- Web3Forms пересылает содержимое формы на email, **привязанный к этому ключу**.
- После отправки пользователя перекидывает на страницу «Спасибо» (`/thanks/` на его языке).
- Есть honeypot (скрытое поле `botcheck`) от ботов.
- Ключ берётся из переменной окружения `PUBLIC_WEB3FORMS_KEY` (ключ публичный — это нормально,
  он и так виден в HTML; в переменной — чтобы легко менять и не коммитить).

## Шаги настройки (один раз)
1. Откройте **https://web3forms.com** → введите email, куда хотите получать заявки
   (например `info@teranovagroup.com`) → получите **Access Key** (строка-идентификатор). Бесплатно, без аккаунта.
2. Cloudflare Dashboard → **Pages** → проект сайта → **Settings → Environment variables**.
3. **Add variable:**
   - **Name:** `PUBLIC_WEB3FORMS_KEY`
   - **Value:** ваш ключ
   - добавьте и для **Production**, и для **Preview**.
4. **Save** → раздел **Deployments** → **Retry deployment / Redeploy** последнего деплоя
   (чтобы ключ попал в новую сборку).
5. **Проверка:** откройте сайт → заполните форму «Под заказ» → «Отправить»:
   - должно перекинуть на страницу `/thanks/`;
   - письмо должно прийти на ваш email (в первый раз проверьте папку **«Спам»**).

## Что приходит и куда
- **Тема письма:** «Заявка "Под заказ" — Teranova» (у формы тендеров — своя тема).
- **Поля заявки:** компания, контактное лицо\*, email\*, телефон, что нужно\*, объём, сроки,
  комментарий (\* — обязательные).
- **Адрес получателя** задаётся НЕ в коде, а в кабинете Web3Forms (привязан к ключу). Чтобы
  сменить почту получателя — поменяйте в Web3Forms или заведите новый ключ и обновите переменную.

## Лимиты и опции
- Бесплатно: ~**250 писем/месяц**. Для старта достаточно; при росте — платный план.
- Можно дополнительно подключить: автоответ клиенту, капчу (reCAPTCHA/hCaptcha), свой SMTP, вебхуки.
- Файлы форм в коде: `src/components/home/OnDemand.astro` (заявка),
  `src/components/pages/Tenders.astro` (тендеры). Менять разметку без отдельной задачи не нужно.

## Если письма не приходят
1. Проверьте, что `PUBLIC_WEB3FORMS_KEY` задан в Cloudflare и сделан **redeploy** после этого.
2. Проверьте папку **«Спам»**.
3. На сайте: правый клик → «Просмотр кода страницы» → найдите `name="access_key"` —
   значение НЕ должно быть пустым.
4. Проверьте статус ключа/почты в кабинете Web3Forms.
