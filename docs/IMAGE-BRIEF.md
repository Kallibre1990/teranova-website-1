# ТЗ по изображениям сайта (для загрузки своих картинок/фото)

Куда класть файлы: папки `src/assets/...` (указаны ниже).
Как заливать: на GitHub откройте нужную папку → **Add file → Upload files** → перетащите файлы → Commit.
Формат загрузки: **JPG или PNG** (можно WebP). Оптимизацию (WebP + размеры под экраны)
я делаю автоматически при сборке — грузите качественный исходник указанного размера **или крупнее**.

## Правила (важно для доверия)
- **Настоящее — только настоящее.** Всё, что намекает «мы были на этом заводе / это наш
  поставщик / наш визит / наша команда» — **только реальные снимки**. Сток/AI здесь нельзя:
  для бизнеса на доверии фейковое «мы там были» бьёт по доверию.
- **Декоративные фоны направлений** (настроение ниши) — можно иллюстративные/лицензионные/AI,
  но нейтральные, без конкретного «реального» поставщика. Это ваше решение как владельца.
- **Без текста на самой картинке** — подписи добавляются кодом.
- **Без «Бусан»** на вывесках/табличках в кадре (правило бренда — «только Корея»).
- **Без чужих логотипов/брендов** крупным планом; лица посторонних — по согласию или нерезко/со спины.
- Тон — холодный/нейтральный (синие, стальные) — лучше ложится на тёмный бренд сайта.

## Технические требования (для всех фото)
- **Формат:** JPG (профиль sRGB). Можно PNG/WebP. Грузите качественный исходник **указанного
  размера или крупнее** (например, ×2) — я сам пересожму в WebP и сделаю размеры под экраны.
- **Вес исходника:** ориентир до ~1–2 МБ — не критично, сожму при сборке.
- **Резкость/свет:** в фокусе, без смаза и сильного шума; ровный свет, без жёлтого пересвета.
- **Ориентация:** горизонтальная (кроме портретов команды — вертикаль).
- **Кадрирование на сайте:** фото обрезается «по центру» до нужного соотношения
  (object-fit: cover). Поэтому снимайте/экспортируйте **в указанном соотношении** и держите
  главный объект по центру, оставляя **~10 % «воздуха» по краям** (края могут срезаться).

## 2. Блок «О нас» на главной — `about-home.jpg`  📷 только реальное · ✅ слот подключён

- **Где на сайте:** главная, блок «О нас» — правая половина рядом с текстом (на телефоне — над

- **Что снять (один сильный «доверительный» момент), варианты:**
  1. Сотрудник Teranova на корейском заводе/у поставщика — разговор с представителем,
     рукопожатие или обсуждение у производственной линии.
  2. Сотрудник осматривает продукцию/образцы прямо на производстве.
  3. Небольшая сцена встречи/переговоров на фоне цеха.

- **Приёмка:** реальное фото · виден человек и контекст (завод/поставщик) · резко · 4:3 ·

## Как загрузить (по шагам)
1. На GitHub откройте нужную папку (напр. `src/assets/verify/`).
2. **Add file → Upload files** → перетащите файл.
3. Имя файла — **точно** как в ТЗ (`site-visit.jpg` и т.п.), иначе сайт его не подхватит.
4. **Commit changes** — при сборке фото оптимизируется и появится на странице.

Если реального фото пока нет — лучше **оставить слот пустым** (стоит аккуратный плейсхолдер),
чем ставить сток. Как появится реальный визит — снимите по этому ТЗ.
