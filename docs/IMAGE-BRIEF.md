# ТЗ по изображениям сайта (для загрузки своих картинок/фото)

Куда класть файлы: папки `src/assets/...` (указаны ниже).
Как заливать: на GitHub откройте нужную папку → **Add file → Upload files** → перетащите файлы → Commit.
Формат загрузки: **JPG или PNG** (можно WebP). Оптимизацию (WebP + размеры под экраны)
я делаю автоматически при сборке — грузите качественный исходник указанного размера **или крупнее**.

## Правила (важно для доверия)
- **Настоящее — только настоящее.** Всё, что намекает «мы были на этом заводе / это наш
  поставщик / наш визит / наша команда» — **только реальные снимки**. Сток/AI здесь нельзя:
  для бизнеса на доверии фейковое «мы там были» бьёт по доверию.
- **Декоративные фоны направлений** (настроение ниши) — можно иллюстративные/лицензионные/AI,
  но нейтральные, без конкретного «реального» поставщика. Это ваше решение как владельца.
- **Без текста на самой картинке** — подписи добавляются кодом.
- Тон — холодный/нейтральный (синие, стальные) — лучше ложится на тёмный бренд сайта.
- Важный объект держите по центру/равномерно; верх и левый край часто перекрыты текстом.
