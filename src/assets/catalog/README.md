# Catalog card background images

Drop a real image here, named by category key — it becomes that card's background
(auto-optimised to WebP). Nothing renders until a file exists, so the cards stay
clean until you provide pictures.

Accepted: `.jpg` `.jpeg` `.png` `.webp`. Names:

- `marine`     — судовое снабжение
- `cosmetics`  — косметика и уход
- `medical`    — медицина и эстетика
- `industrial` — промышленное оборудование
- `transport`  — коммерческий и спецтранспорт
- `chemical`   — химия и материалы
- `adjacent`   — смежные направления

Note: the brand rule (docs/IMAGERY.md) is "no stock/AI imagery". Using AI-generated
or stock pictures here is an explicit owner decision — only add images you are happy
to publish on a trust-first site.
