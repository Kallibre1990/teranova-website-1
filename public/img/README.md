# /public/img — фоновые фото (deep layer)

Сюда кладём реальные фото для глубокого фонового слоя. Они автоматически
приводятся к duotone (navy+серебро), затемняются и ставятся ЗА текстом.
Если файла нет — секция просто показывает сгенерированную тёмную «ткань»
(`tex-deep.webp`), ничего не ломается.

Ожидаемые имена файлов (jpg или webp, лучше ширина ~1600–2000px):

| Файл | Где показывается | Что подойдёт |
|------|------------------|--------------|
| `hero.jpg` | Герой (первый экран) | танкер/море/порт сверху |
| `industry.jpg` | Категории | завод / производственная линия / робот |
| `factory.jpg` | «Под заказ» | цех / автоматизация / станки |

`tex-deep.webp` — сгенерированная «ткань», не трогать (пересоздаётся
`node scripts/make-textures.mjs`).

## Как добавить фото (с телефона/компьютера)
GitHub → ветка PR #4 → папка `public/img` → **Add file → Upload files** →
перетащить файлы с нужными именами → Commit. Превью пересоберётся само.
